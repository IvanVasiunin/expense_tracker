# For developers

In order to run the project:

- Clone the project to your computer
- Install all dependencies with: <code>npm install</code>
- To start your project, use the command: <code>npm start</code>

## Stack and Tools

- HTML5
- CSS3
- Javascript
- React.js (useState)
